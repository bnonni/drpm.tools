## Development

* Clone the repo

```sh
git clone https://github.com/bnonni/drpm.tools
```

* Install dependencies

```sh
cd drpm.tools
npm install
```

* Build

```sh
npm run build
```

* Start registry

```sh
npm run registryd:start
```

* Install cli (optional)

```sh
npm run cli:install-local
```
